# ✅ Task Manager

App per la gestione delle attività quotidiane, costruita con **HTML, CSS e JavaScript vanilla**.

🔗 **Live:** [tuousername.github.io/task-manager](https://tuousername.github.io/task-manager)

---

## ✨ Funzionalità

- Aggiungi task con un click o premendo **Invio**
- Segna i task come completati con un click
- Elimina i singoli task o tutti i completati in un colpo
- Filtri: **Tutti / Da fare / Completati**
- Statistiche in tempo reale (totale, completati, da fare)
- **I dati persistono** tra le sessioni grazie a `localStorage`

## 🛠 Tecnologie

| Tecnologia | Utilizzo |
|---|---|
| HTML5 | Struttura della pagina |
| CSS3 | Layout, animazioni, design responsivo |
| JavaScript | Logica dell'app, gestione del DOM, localStorage |

## 💡 Concetti JavaScript usati

- `localStorage.getItem` / `setItem` per la persistenza dei dati
- `Array.map`, `filter`, `unshift` per la gestione della lista
- `IntersectionObserver` per animazioni
- Sanificazione input con `escapeHtml` per prevenire XSS
- Delegazione eventi
